# تحديث قالب PDF — Doc Math DZ

## الملفات المعدّلة / الجديدة (انسخها فوق مشروعك بنفس المسارات)

1. `src/lib/pdf/exam-template.ts`  ← القالب الجديد كاملاً
2. `src/lib/pdf/generate.ts`       ← هوامش أضيق (النص يملأ الصفحة من اليسار إلى اليمين)
3. `public/images/pdf-cover.png`   ← صورة الغلاف الأول (الصفحة الأولى)
4. `public/images/pdf-back.png`    ← صورة الصفحة الأخيرة

## ما الذي تغيّر

### 1) الصفحة الأولى (للملف الجماعي)
- أصبحت صورة الغلاف التي ولّدتها (DOC MATH DZ) بكامل الصفحة.
- تُحمّل من: `https://www.docmathdz.dev/images/pdf-cover.png`

### 2) الصفحة الأخيرة
- صورة رسالة المنصة (Partager le savoir…) بكامل الصفحة بدل الغلاف الخلفي النصي القديم.

### 3) فهرس Table des matières
- تصميم جديد بنفس روح الصورة: عنوان كبير كحلي بخطوط ذهبية متدرجة،
  سنوات بلون كحلي وخط ذهبي، نقاط ذهبية بين العنوان ورقم الموضوع (مثل فهرس الكتب).

### 4) ترويسة صفحة الامتحان — مصغّرة
- نفس الأسطر الرسمية (الجامعة / Faculté / Concours / السنة / المدة)
  لكن بخط صغير أنيق (7.5–11pt بدل 8.5–15.5pt) + خط فاصل كحلي/ذهبي.
- تأخذ الترويسة الآن مساحة أقل بكثير — مساحة أكبر للتمارين.

### 5) النص يملأ الصفحة كاملة
- إزالة الإزاحة في بداية الفقرات (text-indent) + محاذاة justify كاملة.
- تقليص الهوامش: يمين/يسار 14mm ← 11mm، أعلى 15→13mm، أسفل 16→14mm.

## خطوات التطبيق
1. انسخ الملفات الأربعة إلى مشروعك بنفس المسارات.
2. `git add` ثم انشر على Vercel.
3. مهم: صور الغلاف تُقرأ من الموقع المنشور (أو NEXT_PUBLIC_SITE_URL)،
   فلن تظهر محليًا إلا بعد رفع الصور للموقع أو ضبط NEXT_PUBLIC_SITE_URL=http://localhost:3000.
