# 📝 ميزة «ملاحظاتي» — للمدير الأعلى فقط

صفحة خاصة للدراسة والمراجعة داخل لوحة الإدارة، متّصلة مباشرة بقاعدة **mylibrary** القديمة (MongoDB Atlas) — فتظهر كل دفاترك وملاحظاتك القديمة تلقائيًا دون أي استيراد.

## الرابط

`/admin/notes` — يظهر تبويب «📝 ملاحظاتي» في لوحة الإدارة للمدير الأعلى (SUPER_ADMIN) فقط. أي مستخدم آخر (حتى ADMIN العادي) يُعاد توجيهه، وكل الإجراءات في الخادم محمية أيضًا بفحص الدور.

## الملفات المضافة / المعدّلة

| الملف | الدور |
| --- | --- |
| `src/lib/mylibrary.ts` | جديد — الاتصال بقاعدة mylibrary + تطبيع الحقول القديمة |
| `src/app/admin/notes/actions.ts` | جديد — إجراءات الخادم (إنشاء/تعديل/حذف الدفاتر والملاحظات) |
| `src/app/admin/notes/page.tsx` | جديد — الصفحة (حماية SUPER_ADMIN + تحميل البيانات) |
| `src/app/admin/notes/notes-client.tsx` | جديد — الواجهة الكاملة (3 ألواح) |
| `src/app/admin/layout.tsx` | تعديل — إضافة تبويب «📝 ملاحظاتي» |
| `src/app/globals.css` | تعديل — تنسيق القراءة `.note-reading` |
| `.env.local.example` | تعديل — توثيق المتغيرات الجديدة |

## المزايا

- **استرجاع كل القديم**: يقرأ الدفاتر من `notebooks/notebook/notefolders` والملاحظات من `notes/note/mynotes/usernotes/studynotes`، ويفهم أسماء الحقول المختلفة (`title/name`، `content/body/text/description`، `notebookId/notebook/folderId/folder`).
- **دفاتر**: إنشاء، إعادة تسمية، حذف (مع خيار حذف الملاحظات أو نقلها إلى «غير مصنّفة»)، ألوان مميزة وعدّادات.
- **ملاحظات**: إنشاء، تحرير، حذف، تثبيت 📌، نقل بين الدفاتر، بحث فوري في العناوين والمحتوى.
- **حفظ تلقائي سريع**: يحفظ أثناء الكتابة (كل 0.8 ثانية من التوقف) + `Ctrl+S` للحفظ الفوري، مع مؤشر حالة الحفظ.
- **قراءة مريحة وجميلة**: عرض Markdown كامل + معادلات LaTeX عبر KaTeX (بما فيها صيغة GitLab القديمة <code>$`...`$</code> و<code>```math</code>)، تكبير/تصغير الخط A+/A-، دعم RTL وLTR تلقائي، وضع ليلي/نهاري.
- **محرر متقدم**: نفس محرر LaTeX الموجود في المنصة + معاينة فورية جنبًا إلى جنب.
- **لوحة استكشاف**: أسفل الشريط الجانبي تجد قائمة بكل مجموعات mylibrary وعدد مستنداتها — لتتأكد أين توجد بياناتك القديمة.

## التشغيل

1. متغير `DATABASE_URL_MYLIBRARY` موجود أصلًا في `.env` المحلي. **للنشر: أضفه في Vercel → Settings → Environment Variables.**
2. تأكد في MongoDB Atlas → Network Access أن `0.0.0.0/0` مسموح (أو عناوين Vercel).
3. `npm run dev` ثم افتح `/admin/notes` بحساب المدير الأعلى.

> ℹ️ إن لم تظهر ملاحظاتك القديمة: افتح لوحة «🔎 مجموعات قاعدة mylibrary» في الشريط الجانبي، وابحث عن اسم المجموعة التي تحوي ملاحظاتك، ثم أضف في `.env`:
>
> ```
> MYNOTES_NOTES_COLLECTION="اسم_المجموعة"
> MYNOTES_NOTEBOOKS_COLLECTION="اسم_مجموعة_الدفاتر"
> ```

## ملاحظات أمان

- الكتابة تتم فقط في مجموعات الملاحظات/الدفاتر المسموح بها — لا تلمس الميزة إطلاقًا `doctorateproblems` أو أي مجموعة أخرى.
- يُفضّل لاحقًا تغيير كلمة مرور مستخدم قاعدة البيانات لأنها ظهرت في محادثات/رسائل سابقة.
